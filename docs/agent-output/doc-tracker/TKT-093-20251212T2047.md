## Technical Notes

The `formatDateWithTimezone()` function:
- Uses the browser's `toLocaleString()` API to get timezone information
- Displays dates in the user's **local timezone** (not UTC as the PM decision initially stated)
- Format: `{date} at {time} {timezone}` (e.g., "Dec 15, 2025 at 12:00 AM PST")
- Extracts timezone abbreviation by splitting the time string and taking the last element
